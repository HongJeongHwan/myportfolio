■ 1. 로그인

     : 우측상단의 로그인 버튼을 통해 로그인

     : 계정이 없다면 로그인 버튼클릭후 계정 생성

     : 계정생성시 이메일주소를 실제 메일을 받을 수 있는 이메일을 적어주세요.

       모델이 예측후 결과를 이메일로 전송해 드립니다.

■ 2. 게시판 리스트

     : 화면하단의 요청등록 버튼 클릭

■ 3. 요청 등록

     : 제목, 내용에 적당한 내용을 입력후

     : #3에서 다운로드 받은 이미지를 선택

     : 요청등록 버튼 클릭

■ 4. 결과 확인

     : 게시판 리스트 화면에서 등록한 제목옆에 1이 표시되면 답변이 등록된 것

     : 1) 제목을 클릭해서 예측결과 확인

     : 2) 회원가입시 등록한 이메일에 접속후 예측결과와 동일한 내용의 메일이 왔는지 확인
